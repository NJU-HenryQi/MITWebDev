# HTML

#### Basic Tags
- \<html>: Root of HTML Document
- \<head>: Info about Document
- \<body>: Document Body
- \<h1>\<h2>\<h3>...: Header Tags
- \<p>: Paragraph Tag
- \<div>: Generic Block Section Tag
- \<span>: Generic Inline Section Tag
- \<ol>: Ordered List
- \<ul>: Unordered List
  - \<li>: List Item
- \<hr>: Dividing Line

#### HTML Attributes
    <tagname attriName = value>
e.g:

    <a href="http://weblab.mit.edu">
        Link to web.lab!
    </a>
e.g:

    <img src="xxx.gif" />
